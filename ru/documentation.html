Что такое CMintS?

CMintS - это CMS и статический контент-генератор, который был создан с учетом интернационализации. CMintS довольно простой в установке, он имеет лишь несколько требований.

Note: CMintS is under extensive beta development. Contributions in form of Bugreports, Documentation updates and Content Translations will help to boost the development speed up. Thanks for stars and word spreading.❤️

Установка CMintS

Installing CMintS is quite easy, ensure that you have Node.js installed and run the commands below:

npm install -g cmints

Пример проекта

Для того чтобы быстрее освоится рекомендуется создать пример проекта, запустив:

# Download example project into current directory and build
cmints --example

# Replace optional {PATH} with the path to the download target directory
cmints {PATH} --example

Пример проекта - это содержимое текущего веб-сайта, которое также можно найти в github .

Запустить сервер

For the production:

# Run http server serving current folder
cmints --start

# Replace optional {PATH} with the path to the folder you wish to serve.
cmints {PATH} --start

# https server: Replace {PATH} with the path to the folder you wish to serve
# Replace {PRIVATE_KEY} with the path to the private key file
# Replace {CERTIFICATE} with the path to the certiface file
cmints {PATH} --start --https -k {PRIVATE_KEY} -c {CERTIFICATE}

For development:

# Run http server serving current folder Use --no-cache flag to disable the caching
cmints --start --no-cache

# Replace optional {PATH} with the path to the folder you wish to serve
cmints {PATH} --start --no-cache

Создание статического содержимого

Выполните команды ниже для создания статического содержимого

# Generate static content of current directory
cmints --static

# Replace optional {PATH} with the path to the project for content generation
cmints {PATH} --static
